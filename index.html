Merak ettikleriniz

Karbon kredisi nedir? +
Karbon kredisi, 1 ton CO₂ emisyonunun azaltılmasını veya dengelenmesini temsil eden ticarete konu bir sertifikadır.
Sertifika nasıl alınır? +
Karbon ayak izinizi hesapladıktan sonra size uygun paketi seçip ödeme yaparak anında dijital sertifikanızı alabilirsiniz.
Hangi ödeme yöntemleri kabul ediliyor? +
Kredi kartı, banka havalesi ve kripto para ile ödeme yapabilirsiniz.
API entegrasyonu nasıl yapılır? +
API dokümantasyonumuzu inceleyerek hızlıca entegrasyon yapabilirsiniz. Kurumsal destek ekibimiz size yardımcı olmaya hazır.
